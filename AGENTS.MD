# 项目概览  

React + vite + React Router 框架  
ui组件使用 shadcn/ui，符合简洁大方的风格，并且移动端适配。    
ffmpeg.wasm 用于视频处理  
视频处理试用 webworker 进行，避免阻塞主线程。   

## 功能  

- 视频处理功能 （如：视频剪辑、视频合并、视频转换等）  
- 批量处理功能 （如：批量处理多个视频）    
## 页面  

首页 首页进行项目介绍，项目功能，项目技术栈等。有一个试用区域，用户可以在试用区域进行视频处理。  

视频处理页面 视频处理页面进行视频处理，用户可以在视频处理页面进行视频剪辑、视频合并、视频转换等操作。   
批量任务页面 批量任务页面进行批量处理，用户可以在批量任务页面进行批量处理多个视频。
